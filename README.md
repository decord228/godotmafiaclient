# 🎭 Yamiko Mafia UI (Web Panel)

![Godot Engine](https://img.shields.io/badge/Godot-4.6-%23478cbf?logo=godot-engine&logoColor=white)
![WASM](https://img.shields.io/badge/Platform-Web%20%2F%20WASM-orange)
![License](https://img.shields.io/badge/License-Proprietary-red)

**Yamiko Mafia UI** — это высокотехнологичный веб-интерфейс для ведущих игры «Мафия», разработанный на движке **Godot 4.6**. Панель обеспечивает полный контроль над игровым процессом, визуализируя действия игроков и автоматизируя рутинные задачи ведущего.

---

## ✨ Особенности интерфейса

* **⚡ Динамический UX:** Почти все элементы интерфейса генерируются кодом на лету, что обеспечивает гибкость и высокую скорость работы.
* **🎨 Плавные анимации:** Кастомная система переходов и появлений окон через Tweens для создания ощущения "дорогого" продукта.
* **🌀 Адаптивный Loader:** Стилизованный экран загрузки с имитацией процессов рассылки карт и синхронизации с Discord API.
* **🛰️ Прямая интеграция:** Работает через REST API с Node.js бэкендом, обеспечивая мгновенную обратную связь.
* **📋 Умный Roster:** Автоматическое обновление статусов игроков (жив/мертв, говорит, фолы, участие в голосовании) в реальном времени.

---

## 🛠 Основные модули клиента

### 🔑 Авторизация (Auth)
* Защищенный вход по сессионным токенам.
* **Web Fix:** Исправлена проблема системного буфера обмена (Ctrl+V работает на любой раскладке в браузере).
* Защита от брутфорса с отображением времени блокировки.

### 🏠 Главное меню (Lobby)
* **Импорт из Discord:** Загрузка списка участников напрямую из голосового канала.
* **Конфигуратор:** Настройка ролевого пула, количества мафии и лимитов чата.
* **Reconnect System:** Автоматическое обнаружение активной сессии — возможность продолжить игру или сбросить её после перезагрузки страницы (F5).

### 🎮 Игровая панель (Game Desk)
* **Таймер:** Двухфазный таймер обсуждения (Подготовка + Речь) с автоматическим переходом.
* **DJ Модуль:** Управление музыкальным сопровождением и громкостью бота в войсе.
* **Терминал:** Интерактивный лог событий с BBCode-подсветкой (убийства, проверки, лечение).
* **Admin Modal:** Принудительное завершение игры с выбором победителя.

## 🚀 Техническая информация

* **Движок:** Godot 4.3 (Forward+ / Compatibility).
* **Экспорт:** Web (HTML5 / WebAssembly).
* **Безопасность:** Передача Bearer-токена в заголовках каждого запроса.
* **Оптимизация:** Минимизация количества `HTTPRequest` узлов и использование пула объектов для плашек игроков.

---

## 📦 Деплой на GitHub Pages

1.  Соберите проект в Godot (Web Export).
2.  Включите `coi-serviceworker.js` для поддержки `SharedArrayBuffer`.
3.  Настройте GitHub Pages на корень репозитория.
